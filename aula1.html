O que é HTML?

É uma linguagem de computador que compõe a maior parte das páginas da internet e dos aplicativos online. Um hipertexto é um texto usado para fazer referência a outros textos, enquanto uma linguagem de marcação é composta por uma série de marcações quedizem para os servidores da web qual é o estilo e a estrutura de um documento.
